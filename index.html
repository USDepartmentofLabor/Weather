Weather and PerceivedTemperature Classes

This class is used to access NOAA data for the OSHA Heat Safety App (v.2 prototype). The object makes an hour-by-hour seven day forecast available in an array of structs. The PerceivedTemperature class can be used independently of the Weather class. However, PerceivedTemperature is a dependency of Weather.

Important: This SDK was built using pre-release technology. This SDK is in beta until it has been successfully tested with a production release version of Apple's Swift language. Because of this, if Apple makes any changes to the underlying language, tools, and/or compiler, this SDK may break until fixes are made.

Seven Day Forecast Data

sevenDayForecast[n]

The seven day forecast is an array of structs. The daily forecast struct is defined below. As you can see, each are arrays of certain data types with each array element representing an hour in the day.

Daily forecast properties

Name Data Type Description
forecastTime Array[NSDate] The date and time, by hour
dewPoint Array[Dictionary<String,Int>] Array of dictionary key-value pairs (F, C)
heatIndex Array[Dictionary<String,Int>] Array of dictionary key-value pairs (F, C)
windSpeed Array[Dictionary<String,Int>] Array of dictionary key-value pairs (MPH, KPH)
cloudAmount Array[Int] Cloud coverage in percent
probabilityOfPrecipitation Array[Int] Percent chance of precipitation
humidity Array[Int] Relative humidity
windDirection Array[Int] Direction of the wind in degrees
temperature Array[Dictionary<String,Int>] Array of dictionary key-value pairs (F, C)
windGust Array[Dictionary<String,Int>] Array of dictionary key-value pairs (MPH, KPH)
windChill Array[Dictionary<String,Int>] Array of dictionary key-value pairs (F, C)
windChill Array[Dictionary<String,Int>] Wind chill based on the wind gust. Array of dictionary key-value pairs (F, C)
weatherConditions Array[Dictionary<String,String>] Array of two dictionary key-value pairs.
maxHeatIndex Int Daily maximum heat index
minWindChillF Int Daily minimum wind chill in Fahrenheit
minWindChillC Int Daily minimum wind chill in Celsius
